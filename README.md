## Overview
This repository contains an **end-to-end Machine Learning project** focused on predicting the resale price of used cars using historical automotive data. The project demonstrates a structured ML workflow including data preprocessing, exploratory data analysis, feature engineering, model training, and evaluation.

The project is designed to be **resume-ready**, **interview-friendly**, and **scalable for future enhancements**.

---

## Business Objective
- Predict car prices with high accuracy using regression models  
- Identify the most influential factors affecting resale value  
- Apply practical machine learning concepts to a real-world dataset

## Dataset Information
The dataset includes key car attributes such as:
- Year of manufacture
- Mileage / usage indicators
- Car brand and model details
- Selling price (target variable)

These features are used to train regression models capable of estimating market value effectively.
